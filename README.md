## 🚀 About The Project

NeonTac Pro is a modern Cyberpunk-themed Tic-Tac-Toe game developed using HTML, CSS, and JavaScript.

Unlike a basic Tic-Tac-Toe game, this project includes multiple difficulty levels, smart AI gameplay, animated countdown screens, win/lose popups, and a futuristic neon-inspired user interface.

The goal of this project was to create a visually attractive and interactive browser game while improving frontend development and JavaScript logic skills.

---

## ✨ Features

✅ Player vs Player Mode

✅ Player vs Computer Mode

✅ Easy, Medium, and Hard Difficulty Levels

✅ Smart AI Opponent

✅ 3 → 2 → 1 → START Countdown Animation

✅ Win / Lose / Draw Screens

✅ Cyberpunk Animated Background

✅ Glassmorphism UI Design

✅ Restart Game Functionality

✅ Responsive Design

✅ Interactive User Experience

## 🎮 Gameplay Flow

1. Select Difficulty
2. Choose Game Mode
3. Countdown Starts
4. Match Begins
5. Win, Lose, or Draw Result Screen
6. Restart or Return to Main Menu

## 🧠 AI Logic

The computer opponent:

* Blocks player winning moves
* Attempts winning combinations
* Prioritizes center position
* Prefers corner positions
* Makes strategic decisions instead of random moves
